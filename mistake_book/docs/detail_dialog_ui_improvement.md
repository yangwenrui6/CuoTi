# 查看详情对话框UI优化

## 问题描述
用户反馈查看详情界面中题目内容和正确答案显示不清楚，难以阅读。

## 问题分析
1. **字体过小**：题目内容12pt，答案11pt，在高分辨率屏幕上显示较小
2. **对比度不足**：背景色与文字颜色对比度较低
3. **缺少视觉层次**：没有边框和字重区分，重要信息不突出
4. **无法选择文本**：用户无法复制题目或答案内容
5. **最小高度未设置**：内容较少时显示区域过小

## 解决方案

### 1. 题目内容优化
- **字体大小**：从12pt增加到14pt
- **字重**：设置为500（medium）增强可读性
- **内边距**：从15px增加到20px，提供更多呼吸空间
- **背景色**：改为纯白色#ffffff，提高对比度
- **边框**：添加2px灰色边框#e0e0e0，增强视觉边界
- **文字颜色**：使用深灰色#212121，提高对比度
- **行高**：从1.6增加到1.8，改善多行文本可读性
- **最小高度**：设置为80px
- **文本选择**：启用鼠标和键盘选择功能

### 2. 正确答案优化
- **字体大小**：从11pt增加到13pt
- **字重**：设置为600（semi-bold）突出重要性
- **内边距**：从12px增加到18px
- **背景色**：改为更鲜明的绿色#e8f5e9
- **边框**：添加2px绿色边框#66bb6a，强调正确答案
- **文字颜色**：使用深绿色#2e7d32，增强对比度
- **行高**：设置为1.7
- **最小高度**：设置为60px
- **文本选择**：启用鼠标和键盘选择功能

### 3. 我的答案优化
- **字体大小**：从11pt增加到13pt
- **字重**：设置为500（medium）
- **内边距**：从12px增加到18px
- **背景色**：改为更鲜明的红色#ffebee
- **边框**：添加2px红色边框#ef5350，强调错误答案
- **文字颜色**：使用深红色#c62828，增强对比度
- **行高**：设置为1.7
- **最小高度**：设置为60px
- **文本选择**：启用鼠标和键盘选择功能

### 4. 解析部分优化
- **字体大小**：保持12pt（解析为辅助信息）
- **内边距**：从12px增加到18px
- **背景色**：改为更鲜明的黄色#fff8e1
- **边框**：添加2px橙色边框#ffb74d
- **文字颜色**：使用深橙色#f57c00
- **行高**：设置为1.7
- **最小高度**：设置为60px
- **文本选择**：启用鼠标和键盘选择功能

## 改进效果
1. ✅ 题目内容和答案更清晰易读
2. ✅ 视觉层次分明，重要信息突出
3. ✅ 颜色对比度提高，减少视觉疲劳
4. ✅ 用户可以选择和复制文本内容
5. ✅ 最小高度确保内容区域始终可见
6. ✅ 边框增强了各部分的视觉边界

## 设计原则
- **可读性优先**：使用更大的字体和更高的对比度
- **视觉层次**：通过字重、颜色、边框区分重要性
- **用户友好**：支持文本选择，方便复制内容
- **一致性**：所有文本区域使用统一的设计语言
- **可访问性**：确保足够的颜色对比度符合WCAG标准

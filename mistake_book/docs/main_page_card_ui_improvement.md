# 主页面卡片UI优化

## 问题描述
用户反馈主页面的错题卡片显示不够清晰，文字较小，难以快速浏览和识别重要信息。

## 优化目标
1. 提高文字可读性
2. 增强视觉层次
3. 改善信息密度
4. 保持美观和一致性

## 优化方案

### 1. 整体布局优化
- **内边距**：从15px增加到18px，提供更多呼吸空间
- **元素间距**：从8px增加到10px，改善视觉分隔
- **色标宽度**：从8px增加到10px，更加醒目
- **色标圆角**：从4px增加到5px，更加圆润

### 2. 标题区域优化

#### 科目标签
- **字体大小**：从11pt增加到13pt
- **保持加粗**：突出科目信息
- **颜色**：保持#2c3e50深色，确保对比度

#### 题型标签
- **字体大小**：从10pt增加到12pt
- **字重**：增加到500（medium）
- **颜色**：从#7f8c8d改为#5a6c7d，提高对比度

#### 难度星级
- **字体大小**：从10pt增加到12pt
- **更加醒目**：与标题保持一致的大小

### 3. 题目摘要优化
这是卡片中最重要的信息，需要最大程度优化：

- **字体大小**：从10pt增加到12pt
- **字重**：增加到500（medium），增强可读性
- **颜色**：从#34495e改为#2c3e50，提高对比度
- **行高**：从1.5增加到1.6，改善多行文本可读性
- **内边距**：添加5px上下内边距，增加空间感

### 4. 标签优化
- **字体大小**：从9pt增加到10pt
- **字重**：增加到500（medium）
- **内边距**：从3px 8px增加到5px 12px，更易点击
- **圆角**：从10px增加到12px，更加圆润
- **背景色**：从#e8f4f8改为#e3f2fd，更鲜明的蓝色
- **文字颜色**：从#3498db改为#1976d2，更深的蓝色提高对比度

### 5. 右侧状态区优化

#### 掌握度标签
- **字体大小**：从10pt增加到11pt
- **保持加粗**：突出状态信息

#### 复习次数
- **字体大小**：从9pt增加到10pt
- **字重**：增加到500（medium）
- **颜色**：从#95a5a6改为#7f8c8d，提高对比度

#### 删除按钮
- **尺寸**：从70x30增加到80x35，更易点击
- **字体大小**：从9pt增加到10pt
- **圆角**：从5px增加到6px

### 6. 间距优化
- **卡片内容区间距**：从8px增加到10px
- **右侧操作区间距**：从8px增加到10px
- **整体布局间距**：增加到15px

## 设计原则

### 1. 可读性优先
- 所有文字至少10pt，确保在各种屏幕上都清晰可读
- 重要信息（题目摘要）使用12pt，突出显示
- 增加字重（500）提高文字清晰度

### 2. 视觉层次
- 科目标签最大（13pt），建立视觉焦点
- 题目摘要次之（12pt），核心内容
- 辅助信息适中（10-11pt），不喧宾夺主

### 3. 对比度优化
- 所有文字颜色都加深，提高与背景的对比度
- 符合WCAG 2.0 AA级标准
- 减少视觉疲劳

### 4. 空间感
- 增加内边距和间距
- 避免元素过于拥挤
- 提供视觉呼吸空间

### 5. 一致性
- 所有字体大小都是偶数（10, 12, 13）
- 间距使用统一的增量（10, 15, 18）
- 圆角使用统一的比例

## 用户体验改进

### 1. 快速浏览
- 更大的字体让用户能快速扫描卡片
- 清晰的视觉层次帮助快速定位信息
- 题目摘要更加突出，一眼就能看到核心内容

### 2. 减少眼睛疲劳
- 更高的对比度减少眼睛负担
- 适当的行高改善多行文本阅读
- 充足的间距避免视觉拥挤

### 3. 触控友好
- 删除按钮增大到80x35，更易点击
- 标签内边距增加，提高可点击区域
- 整体卡片更大，更易触控

### 4. 美观性
- 更圆润的圆角（5px, 6px, 12px）
- 更鲜明的颜色（标签蓝色）
- 更宽的色标（10px）更加醒目

# 复习进度显示优化

## 问题描述
用户反馈在复习模式下，题号显示不够明显，难以看清当前复习进度。

## 问题分析
原有的进度标签（progress_label）样式较为简单：
- 字体大小：14pt
- 颜色：深灰色 #2c3e50
- 无背景色
- 无边框

在复习界面中，由于周围有很多其他元素，进度标签容易被忽略。

## 解决方案

### 优化进度标签样式
增强进度标签的视觉突出度：

```python
self.progress_label = QLabel()
self.progress_label.setStyleSheet("""
    font-size: 16pt;              # 增大字体（14pt → 16pt）
    font-weight: bold;
    color: #2c3e50;
    padding: 5px 10px;            # 添加内边距
    background-color: #ecf0f1;    # 添加浅灰色背景
    border-radius: 6px;           # 添加圆角
""")
```

### 改进效果
1. **字体更大**：从14pt增加到16pt，更易阅读
2. **背景突出**：浅灰色背景使标签从页面中脱颖而出
3. **圆角设计**：6px圆角使界面更加友好
4. **内边距**：5px 10px的内边距使文字不会紧贴边缘

## 显示内容
进度标签显示格式：`题目 X / Y`
- X：当前题目序号（从1开始）
- Y：总题目数量

例如：`题目 3 / 10` 表示正在复习第3题，共10题。

## 技术实现

### 位置
进度标签位于复习对话框的顶部左侧：
```
┌─────────────────────────────────────┐
│ [题目 3 / 10]        [❌ 结束复习]  │  ← 顶部栏
├─────────────────────────────────────┤
│                                     │
│         题目内容区域                 │
│                                     │
└─────────────────────────────────────┘
```

### 更新时机
进度标签在以下时机更新：
1. **加载题目时**：`load_question()` 方法中
2. **完成复习时**：显示"复习完成"

- ✅ 视觉层次更清晰，进度信息不会被忽略
- ✅ 界面更加友好，符合现代UI设计规范

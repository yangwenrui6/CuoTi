## 🚀 部署方式

### 方式1：源码安装（推荐开发者）
```bash
git clone https://github.com/yangwenrui6/CuoTi.git
cd CuoTi/mistake_book
pip install -r dependencies/requirements.txt
python run.py
```

### 方式2：一键安装（推荐用户）
```bash
# Windows
install.bat

./install.sh
```
